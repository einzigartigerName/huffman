# Huffman

[Huffman Coding](https://en.wikipedia.org/wiki/Huffman_coding) implementation

## Build
```shell
git clone https://github.com/einzigartigerName/huffman.git
cd huffman
cargo build --release
```

## Usage
To encode:
```shell
huffman encode [FILES]
```
To decode:
```shell
huffman decode [FILES]
```
